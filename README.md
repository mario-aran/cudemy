## Commands

`pnpm install`: Install all packages across all workspaces
`pnpm up --latest -r --interactive`: Update all packages across all workspaces
`pnpm add -w <package_name>`: Add package to workspace root
`pnpm add --workspace <scoped_package_name>`: Add scoped package
